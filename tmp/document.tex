	\block{\techicon 1. Motivation \& Limitations}{
		\begin{center}
		\begin{minipage}[c]{0.485\linewidth}
			\begin{itemize}
				\item \textbf{The Problem:} Speech LLMs excel at content understanding but struggle with fine-grained, word-level temporal alignment.
				\item \textbf{Absolute Timestamp Pitfalls:} Existing methods (e.g., Qwen2-Audio) use formats like $<|0.45|>\text{Okay}<|1.10|>\text{let's}$, leading to:
				\begin{itemize}
					\item \textit{Cumulative Error:} Monotonically increasing values cause error propagation.
					\item \textit{Poor Generalization:} Fails on audio longer than training duration (e.g., predicts $>30$s if only trained on $30$s clips).
					\item \textit{Vocabulary Explosion:} $0.01$s precision over $300$s audio requires $\sim$30,000 distinct timestamp tokens.
				\end{itemize}
				\item \textbf{Our Solution:} \textbf{Relative Time Intervals} \& \textbf{Masked Training}, transforming LLMs into ``temporal-aware content understanding machines''.
				Instead of absolute positions, we model the \textbf{time interval} between consecutive words (e.g., $<|0.65|>$ for words at $0.45$s and $1.10$s).
				\begin{itemize}
					\item \textbf{Compact Vocabulary:} Only a limited range (e.g., $0.00$s to $5.00$s) is needed. Any length of audio can be represented cumulatively.
					\item \textbf{Natural Paradigm:} Aligns perfectly with the autoregressive generation of LLMs. Avoids learning unrelated absolute values.
				\end{itemize}
			\end{itemize}
		\end{minipage}\hfill
		\begin{minipage}[c]{0.485\linewidth}
			\begin{center}
				% 替换为论文 Fig 1 (Absolute vs Relative 可视化对比图)

        \block{\techicon 3. Training Strategies \& Fine-Tuning}{
            
            \textbf{1. Hybrid Parameter-Efficient Fine-Tuning:}
            \begin{itemize}
                \item \textbf{Full-param:} Timestamp embedding layer \& LM Head (learn new temporal vocabulary from scratch).
                \item \textbf{LoRA:} Decoder layers (preserve pre-trained linguistic knowledge, save compute).
            \end{itemize}
            
            \vspace{0.5cm}
            \textbf{2. Joint Temporal Alignment Loss:}
            \begin{equation}
                L_{total} = L_{text} + \lambda L_{timestamp}
            \end{equation}
            Dynamic weighting: $\lambda$ starts at $1$ (focus on text) and increments by $1$ per epoch (shift focus to timestamps).
            
            \vspace{0.5cm}
            \textbf{3. Timestamp Masking (Regularization):}
            \begin{itemize}
                \item Randomly replaces previous timestamps with a $<$mask$>$ token (10\% probability from epoch 2).
                \item Prevents over-relying on ground-truth history. Forces the model to \textit{listen to the actual audio pause} to infer time, enhancing robustness against noisy annotations.
            \end{itemize}
        }

	\block{\techicon 6. Conclusion}{
		\begin{minipage}[l]{0.9\linewidth}
			We propose a novel framework addressing core challenges in precise temporal sequence understanding for Speech LLMs.
			\begin{itemize}
				\item \textbf{Beyond simple tagging:} Endows LLMs with fine-grained, aligned multi-modal spatiotemporal understanding capabilities.
				\item \textbf{Highly Scalable:} Compact vocabulary enables robust timestamping for \textit{arbitrarily long audio} without degradation.
				\item \textbf{Mutual Enhancement:} Temporal alignment regularization actively improves standard ASR performance.
